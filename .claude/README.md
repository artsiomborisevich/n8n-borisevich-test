# AI CLI
This folder contains configuration and documentation for CLAUDE Code, an AI-powered development assistant designed to automate and enhance the development workflow for the project.

## Contents

- **CLAUDE.md**: Main documentation file outlining the project overview, tech stack, architectural patterns, coding standards, security rules, and common commands.

- **agents/**: Directory containing markdown files that define specialized AI agents for various tasks:
  - `bash-expert.md`: Agent focused on shell scripting and command-line operations.
  - `docs-writer.md`: Agent specialized in writing and updating documentation.
  - `infrastructure-expert.md`: Agent for infrastructure-related tasks, such as Kubernetes and Helm.
  - `security-reviewer.md`: Agent dedicated to security audits and reviews.
  - `test-expert.md`: Agent for testing strategies and code coverage.

- **secrets/**: Directory for storing sensitive configuration files, credentials, or environment-specific secrets (currently empty).

## Aim

The purpose of this folder is to centralize all CLAUDE Code-related configurations, ensuring consistent AI-assisted development practices. It enables seamless integration of AI agents into the development process, promoting code quality, security, and efficiency while minimizing manual intervention.
